# What Are the Basic Parts of a Computer?

## Motherboard
The motherboard holds all the memory and connectors that are needed to run the computer. It serves as the main circuit board for the computer.

## CPU
Inside the computer case of the motherboard, you will find the CPU which stands for Central Processing Unit. The CPU is a processor that is responsible for executing instructions and performing calculations. It is often referred to as the brain of the computer

The CU is a small square chip that goes into the motherboard's CPU socket. The CU processor speed is measured in gigahertz (GHz) and mega hertz (MHz)

## RAM
Random Access Memory is a temporary storage location for the comptuer's CPU. Anytime the computer needs to access data quickly, it will use RAM

The more RAM you have on your computer, the faster it will run and the more programs you can run at the same time. If you are running low on RAM, your computer will run slower.

RAM is volatile memory, this means that it is lost when the computer is turned off. This is why it is important to save your work on your computer.

## HDD
When you save your files they are stored on the ahrd drive. The dark disk drive, or HDD, is a permanent storage location that is used to store data even when the computer is turned off.

The hard drive is made up of a spinning platter and an arm. The platter is where the data is stroed and the arm is used to read and write data to the platter. When you have a faster hard drive, your computer will boot up faster and your programs will run faster.

## SSD
Another type of data storage would be Solid State Drive or SSD for short. SSD is a non volatile flash memory that can be used in place of a hard drive. SSDs are faster and smaller than hard drives, but hard drives are cheaper and have more storage capacity.

## PSU
The PSU is responsible for converting electricty from the wall outlet into a form the computer can use. It sends the power from the outlet to the motherboard, CU, and other components of the computer.

## GPU and Expansions cards
The video card, also known as the Graphics Processing Unit (GPU), is responsible for rendering visuals on the computer screen.

The sound card is responsible for playing sound on the computer speakers

The network card is responsible for connecting the computer to the internet.

# What Are the Different Types of Internet Service Providers?

## Internet Sercice Provider (ISP)
An ISP is a company that sells access to the global internet network, essentially. They come in three different tiers

### Tier 1
Giant conlgomerate companies are in this tier, which have the infrastructure to handle most (if not all) of their network traffic independently

### Tier 2
Country wide providers, which sometimes rent access to tier 1 networks but can stand fairly well on their own

### Tier 3
Small companies that primarily focus on providing internet to local markets, and rely on larger ISPs to provide their infrastructure

# What Are the Different Types of Tools Professional Developers Use?

## Computers
A computer is the primary development environment. Professional devs often go for computers with fast processing power and plenty of RAM to handle resource-intensive tasks

## IDEs
After the computer is a program for writing and editing code right on that computer. That's a code editor or integrated development environment (IDE)

Some good IDEs:
- Visual Studio
- IntelliJ IDEA
- JetBrains
- PyCharm

## Version Control System
When you write code with editors and IDEs, you need to track the changes you make to them. The tool that lets you track those changes is a version control system.

Git is the most widely used version control system in the development community.

Platforms like:
- GitHub
- GitLab
- Bitbucket
provide cloud-based hosting services for your Git repositories. These platforms enable collaboration with other developers, allow you to work on multiple branches, and facilitate the merging of code changes.

## Package Managers
Package managers are another critical tool. They help devs simplify the process of adding, updating, and removing libraries and project dependencies.

Examples of popular package managers are:
- NPM, Yarn, and PNPM for JavaScript
- PIP for Python
- Composer for PHP
- Maven for Java


## Testing Frameworks
After writing code with differnet tools, devs test that code to make sure it's working as expected.

For this, devs use testing frameworks like Cypress, playwright, Selenium, and others. 

There are also language specific testing frameworks such as:
- Jest for JavaScript
- pytest for Python
- JUnit for Java
- PHPUnit for PHP

# What Are Best Practices for Naming Files for Web Applications?

## Good vs Bad Names
An example of a bad file name is index1234.html
- By the name alone, it is not clear what the file is about. This file could be for a products page, blog page, or any other type of page but we don't know that from the file name

A better file name for an HTML file could be about-us.html
- By using a more descriptive name like about-us.html, it is clear what the file is about. 

It is also better to never use spaces in file names, and instead use hyphens. On that note, it is usually not a good idea to use special characters either.

# What Are Some Common File Types You Will Work With in Web Applications?

## File Types

### General Web

- HTML files contain the structure and content of the web application. They have a .html file extention.

- CSS files define the styles, they have a .css file extention

- JAVAscript add more functionality and interactivity, they have the .js file extention

### Images, Audio, and Video

- JPEG which stands for "Joint Photographic Experts Group? is known for its efficient compression. Its great for photographs and images, however the compression is lossy, which means that some image quality is lost to reduce the file size.

- PNG which stands for "Portable Network Graphics" preserves image quality, even after comrpession. This results in larger files. PNG supports transparency and it's great for images with sharp edges, like logos and icons.

- The GIF image format supports both animation and transparency but has a limited color palette. GIF stands for Graphics Interchange Format.

- SVG stands for Scalabe Vector Graphics. It's a vector image format. These images can be scales as much as needed without losing quality.

- MP3 is an audio format known for its efficient compression. It's a lossy format, so some audio data is lost during compression to make these files smaller.

- MP4 is one of the most common video formats. It offers good compression and supports multiple audio and video codecs, subtitles, and metadad.

- MOV, the QuickTime multimedia file format developed by Apple, is primarily associated with QuickTime Player.

- WAV is a lossless audio format that keeps the original quality of the audio, and then WebM, a high-quality open source video format.

### Fonts

- TTF, which stands for "TrueType Font" is a font format compatible with different operating systems.

- WOFF is a modern font format, specifically designed for web dev purposes. It stands for "Web Open Font Format" These files are smaller because they have better compression. They can also store metadata, including licensing information.

- WOFF2 is the successor of WOFF withe ven more efficient compression and performance

### Archive files
You can also create arhchive files if you need to group multiple folders and files. The most widely used archive format is ZIP

- ZIP offers lossless compression, so no data is lost during the process. It's widely supported across operating systems and software applications.

### Documentation
In web applications, there is usually a file called README that contains useful information about the application. They are usually written in a file format called markdown

- .md or .markdown is a file format that allows you to create structured documents with headings, subheadings, links, images, lists, and more.

# What Is the Difference Between a Web Browser, a Website, and a Search Engine?

## Web browsers
Web browsers are software applications that allow users to access and navigate the World Wide Web. They inrepret and display web content, including text, images, videos, and interactive elements.

## Websits
Websites are a collection of web pages and related content  that are indetified by a common domain name and published on at least one web server. They are destinations you visit using your web browser

## Search Engines
Search engines are web based tools designed to help users find information on the internet. They use complex algorithms to index and rank web pages based on relevance to user queries.

Key aspects of search engines include web crawlers, also known as spiders or bots, which are programs that systematically browse the web to discover and index new content. 

Search engines also employ ranking algorithms, which are complex systems that determine the order of search results based on factors like relevance, authorirty, and user experience.
