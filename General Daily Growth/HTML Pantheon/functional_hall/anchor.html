Each of these links have their target attribute value set to _blank, meaning when you click them, they'll pop up as a new tab instead of in the same window! The Home button however, does not have any target attribute. Which is why when you click it, it'll open up in your current tab instead of a new one.

Each anchor link can also use something called a file path

<a href="example_folder/example_object.html">Example Link</a>

For this code, the anchor element wouuld go into example_folder and then access example_object when the user clicks on Example Link
